Godot version 4.7 dev 1 introduced a new UI node: the [Virtual Joystick](https://godotengine.org/article/dev-snapshot-godot-4-7-dev-1/#input-virtualjoystick).

I was able to add a virtual joystick to the [2D intro tutorial](https://docs.godotengine.org/en/stable/getting_started/first_2d_game/index.html) with very little effort. If you want to try it out, you can play it [here](https://eliasprescott.github.io/godot-demo/) or you can try playing it in the iframe below.
